def metplus.wrappers.runtime_freq_wrapper.RuntimeFreqWrapper.get_all_files (   self)

Get all files that can be processed with the app.

Some wrappers like UserScript do not need to obtain a list of possible files. In this case, the function returns an empty dictionary.

Returns
A dictionary where the key is the type of data that was found, i.e. fcst or anly, and the value is a list of files that fit in that category
