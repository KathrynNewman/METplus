Test that the user's version of python is equal of higher than the the supported version of python.

Imported in each wrapper and run_metplus to avoid confusing failures if the user's version is not current. Note: SyntaxError from using f-strings (available in 3.6+) in earlier versions of Python are output before the output from this function can be displayed. Args:

Parameters
user_pyuser's python version number, i.e. 3.8.1
supported_pycurrently supported python version number, i.e. 3.6.3
Returns
True if version is at least supported, False if not
